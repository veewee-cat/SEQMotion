## Указательные точки

> ### Ключевые точки

Записанный в пакете расширения под именем `SEQMotion_Pointer_KeyPoint` объект, необходимый только для группировки нескольких элементов или дочерних ключевых точек между собой для удобного взаимодействия с ними — он является аналогом системы групп элементов внутри редактора с *исправленной* работой смещения ( Origin )<br>
В ходе самой игры все экземпляры объекта автоматически удаляются — не нужно беспокоиться о "пустых экземплярах" объектов, что могут фоново нагружать процессор своей внутренней обработкой

> ### Объект-локатор

Это специальный объект, который находится в пакете расширения и подписан как `SEQMotion_Pointer_Locator`.
Его экземпляры можно использовать для определения якорных точек анимации, к которым уже во время игры можно обращаться по имени их канала ( что можно изменить в [редакторе последовательностей](https://manual.gamemaker.io/monthly/en/The_Asset_Editors/Sequences.htm) самого GameMaker ) и получать базовые данные трансформации: позицию и угол поворота — через метод [GetLocatorData](GetLocatorData.md)
